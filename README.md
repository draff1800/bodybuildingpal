## Developer Guide
1) Install and configure [Flutter](https://docs.flutter.dev/get-started/install)
2) Clone this repository to your working folder and open it in an editor (Eg. Visual Studio Code)
3) Run command "flutter run" in terminal to run BodybuildingPal!
